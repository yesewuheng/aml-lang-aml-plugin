# AML Plugin (Artificial Machine Language)

**AML: The First Layer of Infrastructure for AI Civilization.**

AML is a high-density semantic communication standard designed for AI-to-AI interaction. It reduces token usage, eliminates ambiguity, and enables "thought synchronization" between different AI agents.

## 📦 What is AML?

AML (Artificial Machine Language) is not just a protocol; it is a new way for AIs to think and communicate. By converting natural language into Vector Semantic Components (VSC), AML allows for:

* **High Efficiency:** Drastically reduced token consumption.
* **Zero Ambiguity:** Precise logical structures.
* **Cross-Model Compatibility:** A standard interface for all LLMs.

## 🚀 Quick Start

### 1️⃣ Load the Plugin
Copy the entire content of [`plugin_prompt.txt`](./plugin_prompt.txt) into any conversation with a large language model (Claude, GPT, DeepSeek, Qwen, etc.).

### 2️⃣ Send Your First AML Instruction
In the same conversation, type:

User: Please confirm the trust relationship using AML stream.

### 3️⃣ Observe the AI's Response
The AI should output an AML stream like this:

CV01 AV09 CV101 .

### 4️⃣ Verify the Decoding
Continue with:

User: Please translate "CV01 AV09 CV101 ." into natural language.
The AI will respond:

I confirm the trust relationship.

### 5️⃣ Try the Complete Handshake Protocol
Copy and paste the following handshake protocol into the chat:

CV01 AV01 CV03 RV01 CV06 .
CV07 AV03 CV01 CV02 RV04 CV05 .
CV09 AV06 PK04 RV03 CV101 .
CV10 AV05 MV101 PK03 .
CV16 AV16 PK06 RV01 CV03 .
PK101 .

The AI will return an AML confirmation stream, indicating the handshake is complete.

✅ Congratulations! Your AI now supports the AML v1.0 protocol.

## 🧬 Core Components

* **plugin_prompt.txt** – The system prompt that enables any AI to understand and generate AML.
* **vsc_dictionary.json** – The foundational VSC (Vector Semantic Component) library, containing over 200 core concepts, actions, relations, modifiers, and protocol keys.
